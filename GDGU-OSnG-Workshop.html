Workshop on Open Source and Git

The workshop is designed to introduce students to the world of open source and how git perfectly fits into the picture.

Aim:

  • To introduce the concept of open source and how it is changing the face of industry
  • To get acquainted with version control systems
  • Learn and use git version control tool
  • Explore GitHub platform features
  • To understand CI/CD

What we'll cover:

  • Purpose and Community
  • Open Source
  • Agile Development Methodologies
  • Version Control Systems
  • Git as a DVCS
  • Integration with Git and GitHub
  • Exploring Further!
